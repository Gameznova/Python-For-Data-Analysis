# Bank Teller

A simple Python project completed as part of Codecademy’s Python for Finance exercises.

## Description

This program simulates basic bank teller transactions between checking and savings accounts.

Users can:

* Check account balances
* Make deposits
* Make withdrawals
* Transfer money between accounts

## Technologies

* Python 3

## Purpose

This project practices Python functions, variables, conditional statements, parameters, return values, and basic account balance calculations.
